# Python-Challenge

Python is a popular language used in data science. In this repository, two projects demonstrate the utility of python in two different fields. 
PyBank shhows how Python can be used to perform operations on and summarize financial and accounting data; It does this by condensing a simple accounting report into summary statistics 
while PyPoll shows how Python can automate the tedious process of sifting through and counting repeated instances as is common in election data. 
Python can be used to analyze any data contained in a csv format without relying on third-party software.
